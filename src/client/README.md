# Quality commands

- `make check` ‚Äî run lint + format check + type check + tests
- `make fix` ‚Äî auto-fix lint issues and format code
- `make lint` ‚Äî run Ruff linter
- `make fmt` ‚Äî format code with Ruff
- `make fmt-check` ‚Äî verify formatting without changing files
- `make typecheck` ‚Äî run mypy
- `make test` ‚Äî run pytest (no tests collected is treated as success)
